### What is a transverse wave?
A wave where the disturbance is **perpendicular** to the direction the wave travels. Think of shaking a rope side-to-side — the rope moves up and down, but the wave travels forward.

A wave moves energy, not matter.

### The core wave equation

Wave equation
y = y$_m$ sin(kx − ωt)
y$_m$ = amplitude, k = wave number, ω = angular frequency

Wave speed
v = λ/T = λf
λ = wavelength, T = period, f = frequency

Wave number
k = 2π/λ

Angular frequency
ω = 2π/T = 2πf

### Wave interference — when two waves meet

Two waves occupy the same space at the same time — they add up. The result depends on their _phase difference (φ)_.

*General result*
y = (2y_m cos(φ/2)) sin(kx − ωt + φ/2)

Amplitude changes by 2y_m cos(φ/2)

*Fully constructive (φ = 0)*
Amplitude = 2y_m

Waves perfectly in phase → maximum

*Fully destructive (φ = π)*
Amplitude = 0

Waves perfectly out of phase → cancel

φ = 0 → double amplitude. φ = π → silence. Anything in between is partial.

### Standing waves

A wave travelling right + the same wave travelling left = a standing wave. The wave doesn't appear to move — it just oscillates in place.

Standing wave equation
y = (2y_m sin(kx)) cos(ωt)
Nodes (no movement) at sin(kx)=0. Antinodes (max movement) at sin(kx)=±1.

==Guitar strings, organ pipes, drum membranes — all standing waves.==
