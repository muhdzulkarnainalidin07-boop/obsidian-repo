### The kinematic equations (constant acceleration only)
Velocity
v = v₀ + at

Position
x = x₀ + v₀t + ½at²

No time
v² = v₀² + 2a(x−x₀)

Average velocity
v_avg = (v₀ + v)/2
These only work when a is CONSTANT. If a changes with time, use calculus.
### Free fall
Constant acceleration downward: g = 9.8 m/s²
At the top of a throw: velocity = 0, but acceleration is still −9.8 m/s² downward. The object is still accelerating even when momentarily still.
### Projectile motion
Treat horizontal and vertical _independently_.

Horizontal (no accel)
x = v₀cos(θ)·t
Constant speed horizontally

Vertical (free fall)
y = v₀sin(θ)·t − ½gt²
Gravity pulls down

Range
R = v₀²sin(2θ)/g
Max range at θ = 45°

Max height
H = v₀²sin²(θ)/2g

### Circular motion
Centripetal acceleration
a_c = v²/r
Always points toward centre

Period
T = 2πr/v
Speed is constant in uniform circular motion, but velocity is not — it's always changing direction.
