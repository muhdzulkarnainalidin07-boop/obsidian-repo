### 9:30–9:55 | Transverse waves: wave equation + extraction drill

#### READ THIS FIRST

**The wave equation:**

$$y(x, t) = y_m \sin(kx - \omega t)$$

Every symbol labelled:
- $y_m$ = amplitude (maximum displacement, metres) — always positive
- $k$ = wave number (rad/m) — **not** the spring constant here
- $\omega$ = angular frequency (rad/s)
- $(kx - \omega t)$ = phase — determines position on the cycle at any given $x$ and $t$
- The **minus** sign: wave travels in the $+x$ direction. $(kx + \omega t)$ would travel in $-x$.

**All derived quantities — know these cold:**

$$\lambda = \frac{2\pi}{k}, \quad T = \frac{2\pi}{\omega}, \quad f = \frac{1}{T} = \frac{\omega}{2\pi}$$

$$v_{\text{wave}} = f\lambda = \frac{\omega}{k} \quad \text{[wave speed — how fast the pattern moves through space]}$$

$$u_{\text{max}} = \omega y_m \quad \text{[maximum transverse speed — how fast a particle of the medium moves up/down]}$$

$$a_{\text{max}} = \omega^2 y_m \quad \text{[maximum transverse acceleration]}$$

**Critical distinction:** $v_{\text{wave}} = \omega/k$ and $u_{\text{max}} = \omega y_m$ are different quantities. Wave speed is the speed of the pattern. Transverse speed is the speed of a particle in the medium moving perpendicular to the wave direction. The exam will test whether you know this.

**The extraction drill — this is the exact question type:**

Given $y(x,t) = 0.04\sin(2.5x - 120t)$ metres:

$$y_m = 0.04\ \text{m}$$

$$k = 2.5\ \text{rad/m} \implies \lambda = \frac{2\pi}{2.5} \approx 2.51\ \text{m}$$

$$\omega = 120\ \text{rad/s} \implies T = \frac{2\pi}{120} \approx 0.052\ \text{s} \implies f = \frac{1}{T} \approx 19.1\ \text{Hz}$$

$$v_{\text{wave}} = \frac{\omega}{k} = \frac{120}{2.5} = 48\ \text{m/s}$$

$$u_{\text{max}} = \omega y_m = 120 \times 0.04 = 4.8\ \text{m/s}$$

$$a_{\text{max}} = \omega^2 y_m = 14400 \times 0.04 = 576\ \text{m/s}^2$$

#### NOW: Close the file. Write $y(x,t)$ and label every symbol. Then make up your own equation and do the full extraction drill — $y_m$, $k$, $\omega$, $\lambda$, $T$, $f$, $v_{\text{wave}}$, $u_{\text{max}}$, $a_{\text{max}}$.

If you can do it cold in under 2 minutes, you are ready.

> **Check pass (9:50):** All quantities extracted correctly? $v_{\text{wave}}$ and $u_{\text{max}}$ are different values using different formulas — did you get both right?

### 10:05–10:25 | Wave on a string + weak spot revisit

#### READ THIS FIRST

**Wave speed on a string:**

$$v = \sqrt{\frac{\tau}{\mu}}$$

- $\tau$ = tension in the string (Newtons) — different from torque $\tau$ in rotational motion
- $\mu$ = linear mass density (kg/m) — mass per unit length

$$\mu = \frac{m}{L} \quad \text{[total mass / total length]}$$

**Unit trap — the most common error in this topic:**

If $\mu$ is given in g/m, convert to kg/m first. $2\ \text{g/m} = 0.002\ \text{kg/m}$. Forgetting this gives a speed off by a factor of $\sqrt{1000} \approx 31.6$. The answer will be obviously wrong in scale — use that as a check.

**Worked example:**

String: tension $\tau = 50\ \text{N}$, mass $m = 10\ \text{g}$ over length $L = 2\ \text{m}$.

$$\mu = \frac{0.010\ \text{kg}}{2\ \text{m}} = 0.005\ \text{kg/m}$$

$$v = \sqrt{\frac{50}{0.005}} = \sqrt{10000} = 100\ \text{m/s}$$

**Power transmitted by a wave on a string:**

$$P_{\text{avg}} = \frac{1}{2}\mu\omega^2 y_m^2 v$$

Know this exists and what variables it contains — it appears in harder problems.
