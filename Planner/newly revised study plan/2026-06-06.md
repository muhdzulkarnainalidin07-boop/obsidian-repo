### 8:00–8:25 | Complex Numbers: Cartesian and polar forms

#### READ THIS FIRST

A complex number $z$ can be written in three equivalent forms:

$$z = a + bi \quad \text{[Cartesian]} \qquad a = \text{Re}(z),\ b = \text{Im}(z)$$

$$z = r(\cos\theta + i\sin\theta) \quad \text{[Polar]} \qquad r = |z|,\ \theta = \arg(z)$$

$$z = re^{i\theta} \quad \text{[Exponential]} \qquad \text{Euler: } e^{i\theta} = \cos\theta + i\sin\theta$$

**Converting Cartesian → Polar:**

$$r = \sqrt{a^2 + b^2} \quad \text{(always positive)}$$

$$\theta = \arctan\!\left(\frac{b}{a}\right) \quad \text{BUT — check the quadrant manually every time}$$

**Quadrant rule — this is where marks are lost:**

| Quadrant | Condition | Correct $\theta$ |
|----------|-----------|-----------------|
| Q1 | $a > 0,\ b > 0$ | $\arctan(b/a)$ — calculator is correct |
| Q2 | $a < 0,\ b > 0$ | $\pi - \arctan(\|b/a\|)$ |
| Q3 | $a < 0,\ b < 0$ | $-\pi + \arctan(\|b/a\|)$ — result is negative |
| Q4 | $a > 0,\ b < 0$ | $\arctan(b/a)$ — calculator gives negative angle, which is correct |

**Worked Q2 example:**

$$z = -3 + 4i$$

$$r = \sqrt{(-3)^2 + 4^2} = \sqrt{9+16} = 5$$

$$\theta = \pi - \arctan\!\left(\frac{4}{3}\right) = \pi - 0.927 \approx 2.214\ \text{rad} \approx 126.87°$$

$$z = 5(\cos 2.214 + i\sin 2.214) = 5e^{2.214i}$$

**Multiplication in polar/exponential form:**

$$z_1 \cdot z_2 = r_1 r_2 \cdot e^{i(\theta_1 + \theta_2)}$$

Multiply the moduli, add the arguments. Much faster than Cartesian multiplication for large powers.

**Verification step:** After multiplying in polar, expand in Cartesian and confirm the results match.

### 8:35–9:00 | Complex Numbers: De Moivre and roots

#### READ THIS FIRST

**De Moivre's theorem:**

$$(re^{i\theta})^n = r^n e^{in\theta}$$

Equivalently:

$$\left(r(\cos\theta + i\sin\theta)\right)^n = r^n(\cos n\theta + i\sin n\theta)$$

**Use for:** Powers of complex numbers. Convert to polar → raise to the power → convert back.

**Finding $n$th roots — the procedure:**

The $n$ distinct $n$th roots of $z = re^{i\theta}$ are:

$$z_k = r^{1/n} \cdot e^{i\frac{\theta + 2\pi k}{n}}, \quad k = 0, 1, 2, \ldots, n-1$$

**What this means geometrically:**
- All roots have the same modulus: $r^{1/n}$
- They are equally spaced around a circle, separated by $\frac{2\pi}{n}$ radians
- The first root ($k=0$) sits at angle $\theta/n$

Always draw the Argand diagram — the equal spacing is what makes this visual, and the visual is what sticks.

**Worked example — cube roots of $8i$:**

$$8i = 8e^{i\pi/2} \quad (|8i| = 8,\ \arg(8i) = \pi/2)$$

$$r^{1/3} = 8^{1/3} = 2, \quad \text{spacing} = \frac{2\pi}{3} = 120°$$

$$k=0:\ 2e^{i\pi/6} = 2\!\left(\cos 30° + i\sin 30°\right) = \sqrt{3} + i$$

$$k=1:\ 2e^{i5\pi/6} = 2\!\left(\cos 150° + i\sin 150°\right) = -\sqrt{3} + i$$

$$k=2:\ 2e^{i3\pi/2} = 2\!\left(\cos 270° + i\sin 270°\right) = -2i$$

**Draw the Argand diagram:** Circle of radius 2, three points at $30°$, $150°$, $270°$. Equally spaced at $120°$ apart. If they are not equally spaced in your drawing, your arithmetic is wrong.

### 9:10–9:35 | Linear Transformations: definition + rotation + reflection

#### READ THIS FIRST

**Definition of a linear transformation** $T: \mathbb{R}^m \to \mathbb{R}^n$:

$$T(\mathbf{u} + \mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v}) \quad \text{[additivity]}$$

$$T(c\mathbf{u}) = cT(\mathbf{u}) \quad \text{[homogeneity]}$$

Combined: $T(a\mathbf{u} + b\mathbf{v}) = aT(\mathbf{u}) + bT(\mathbf{v})$.

Every linear transformation from $\mathbb{R}^n \to \mathbb{R}^m$ can be written as $T(\mathbf{x}) = A\mathbf{x}$ for some $m \times n$ matrix $A$.

**Rotation matrix (anticlockwise by $\theta$):**

$$R(\theta) = \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}$$

**Reflection matrices:**

| Reflect about | Matrix |
|---------------|--------|
| $x$-axis | $\begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$ |
| $y$-axis | $\begin{pmatrix} -1 & 0 \\ 0 & 1 \end{pmatrix}$ |
| $y = x$ | $\begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$ |
| $y = -x$ | $\begin{pmatrix} 0 & -1 \\ -1 & 0 \end{pmatrix}$ |

**Determinant check — write this rule:**

$$\det(R(\theta)) = \cos^2\theta + \sin^2\theta = +1 \quad \text{[rotation — orientation preserved]}$$

$$\det(\text{reflection matrix}) = -1 \quad \text{[orientation reversed]}$$

**Visual check:** Apply $R(90°)$ to the unit square with vertices $(0,0),(1,0),(1,1),(0,1)$:
$(1,0) \to (0,1)$, $(0,1) \to (-1,0)$, $(1,1) \to (-1,1)$.
Draw both the original and rotated square. This is what a rotation matrix does geometrically.

### 9:45–10:10 | Linear Transformations: image finding + composition

#### READ THIS FIRST

**Image of a transformation:**

$$\text{Im}(T) = \{T(\mathbf{x}) : \mathbf{x} \in \text{domain}\} = \text{column space of } A$$

**Finding the image of a specific vector:** Compute $A\mathbf{x}$ directly.

**Finding the image of a spanned set:**
If a set is spanned by $\{\mathbf{v}_1, \mathbf{v}_2, \ldots\}$, its image under $T$ is spanned by $\{A\mathbf{v}_1, A\mathbf{v}_2, \ldots\}$.

**Finding the pre-image (inverse image):**
Given $T(\mathbf{x}) = \mathbf{b}$, find $\mathbf{x}$. Solve $A\mathbf{x} = \mathbf{b}$ via Gaussian elimination.
- Unique solution: exactly one vector maps to $\mathbf{b}$
- Infinitely many: a whole family maps to $\mathbf{b}$
- No solution: $\mathbf{b}$ is not in the image of $T$

**Composition — the order rule:**

"Apply $S$ first, then $T$" → the composed transformation is $T \circ S$, and the matrix is $TS$ (right-to-left):

$$(T \circ S)(\mathbf{x}) = T(S(\mathbf{x})) = (TS)\mathbf{x}$$

Write this before every composition problem: **rightmost matrix acts first.**

**Worked example:**

$S$ = rotation by $90°$, $T$ = reflection about $x$-axis.

$$TS = \begin{pmatrix}1&0\\0&-1\end{pmatrix}\begin{pmatrix}0&-1\\1&0\end{pmatrix} = \begin{pmatrix}0&-1\\-1&0\end{pmatrix}$$

Check: $ST = \begin{pmatrix}0&-1\\1&0\end{pmatrix}\begin{pmatrix}1&0\\0&-1\end{pmatrix} = \begin{pmatrix}0&1\\1&0\end{pmatrix}$

$TS \neq ST$ — order matters.

### 10:20–10:45 | Energy: work-KE theorem + potential energy

#### READ THIS FIRST

**Work-KE theorem:**

$$W_{\text{net}} = \Delta KE = \frac{1}{2}mv_f^2 - \frac{1}{2}mv_i^2$$

Net work done on an object equals its change in kinetic energy.

**Work by a constant force:**

$$W = Fd\cos\theta \quad \text{($\theta$ between force vector and displacement vector)}$$

- Force in direction of motion: $W > 0$
- Force opposing motion: $W < 0$
- Force perpendicular to motion: $W = 0$ (e.g. normal force, gravity on horizontal surface)

**Potential energy — two formulas:**

$$U_g = mgh \quad \text{[gravitational — set } h=0 \text{ at your reference point]}$$

$$U_s = \frac{1}{2}kx^2 \quad \text{[elastic spring — } x \text{ is displacement from natural length]}$$

**The decision you must write before every energy problem:**

1. Which PE applies? Gravity → $mgh$. Spring → $\frac{1}{2}kx^2$. Sometimes both.
2. Is there friction? If yes → use the friction version of conservation. If no → pure conservation.

**Worked example:**

Block (2 kg) released from rest on a spring compressed $x = 0.3\ \text{m}$ ($k = 500\ \text{N/m}$), frictionless surface. Find speed when spring returns to natural length.

$$U_s = \frac{1}{2}(500)(0.3)^2 = \frac{1}{2}(500)(0.09) = 22.5\ \text{J}$$

All PE becomes KE:

$$\frac{1}{2}mv^2 = 22.5 \implies v = \sqrt{\frac{2(22.5)}{2}} = \sqrt{22.5} \approx 4.74\ \text{m/s}$$

#### NOW: Close the file. Write the work-KE theorem, both PE formulas, and the two decision questions. Work the example with your own spring and mass values.

### 10:55–11:20 | Energy: conservation + power

#### READ THIS FIRST

**Conservation of mechanical energy — two versions:**

Without friction:
$$KE_i + PE_i = KE_f + PE_f$$

With friction:
$$KE_i + PE_i = KE_f + PE_f + Q$$

where $Q = f_k \cdot d$ is the thermal energy generated (friction force × distance slid). This is energy that left the mechanical system as heat.

**What friction changes:** Total energy is still conserved — some mechanical energy converts to thermal. The equation accounts for this by adding $Q$ to the right side.

**Finding friction force on an incline:**

$$f_k = \mu_k N, \quad N = mg\cos\theta \quad \text{(on incline at angle } \theta \text{ to horizontal)}$$

On flat ground: $N = mg$.

**Power — two formulas:**

$$P = \frac{W}{t} \quad \text{[work per unit time]}$$

$$P = Fv \quad \text{[when constant force acts on a moving object]}$$

Units: Watts (W) = J/s.

**Worked example with friction:**

2 kg block slides 4 m down a ramp ($h = 2\ \text{m}$, angle $= 30°$, $\mu_k = 0.2$).

$$N = mg\cos 30° = 2(9.8)(0.866) = 16.97\ \text{N}$$

$$f_k = (0.2)(16.97) = 3.39\ \text{N}$$

$$Q = f_k \cdot d = (3.39)(4) = 13.57\ \text{J}$$

$$\Delta PE = mgh = 2(9.8)(2) = 39.2\ \text{J}$$

$$KE_f = 39.2 - 13.57 = 25.63\ \text{J} \implies v = \sqrt{\frac{2(25.63)}{2}} \approx 5.06\ \text{m/s}$$

**Verification:** $Q + KE_f = 13.57 + 25.63 = 39.2 = \Delta PE$ ✓

### 11:30–11:50 | Momentum: collision types + elastic shortcuts

#### READ THIS FIRST

**Momentum:**

$$\mathbf{p} = m\mathbf{v} \quad \text{[vector — direction matters]}$$

**Impulse-momentum theorem:**

$$\mathbf{J} = F\Delta t = \Delta\mathbf{p}$$

**Conservation of momentum** (holds for all collision types):

$$m_1v_{1i} + m_2v_{2i} = m_1v_{1f} + m_2v_{2f}$$

**Elastic vs inelastic — write in plain words before formulas:**

- **Elastic:** both momentum and kinetic energy are conserved. Objects bounce with no energy loss. Real example: billiard balls (approximately).
- **Inelastic:** momentum is conserved, KE is not. Some KE converts to heat/sound/deformation.
- **Perfectly inelastic:** objects stick together. Maximum possible KE loss. Still conserves momentum.

What is "lost" in an inelastic collision: KE converts to internal energy — heat, sound, permanent deformation. Total energy is conserved — just not mechanical KE.

**Elastic collision shortcuts — only valid when target is initially at rest ($v_{2i} = 0$):**

$$v_{1f} = \frac{m_1 - m_2}{m_1 + m_2}v_{1i}$$

$$v_{2f} = \frac{2m_1}{m_1 + m_2}v_{1i}$$

**Special cases — verify these make physical sense:**

Equal masses ($m_1 = m_2$):

$$v_{1f} = 0, \quad v_{2f} = v_{1i}$$

First object stops dead. Second takes all the speed. **You have seen this at the pool table** — cue ball stops, object ball rolls away at cue ball's original speed.

Heavy hits light ($m_1 \gg m_2$):

$$v_{1f} \approx v_{1i}, \quad v_{2f} \approx 2v_{1i}$$

Barely slows. Light object flies off at roughly double the incoming speed.

Light hits heavy ($m_1 \ll m_2$):

$$v_{1f} \approx -v_{1i}, \quad v_{2f} \approx 0$$

Bounces back at nearly same speed. Heavy object barely moves.
