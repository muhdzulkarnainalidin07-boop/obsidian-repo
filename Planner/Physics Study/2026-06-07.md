## Day 4
*Math: [[Curvilinear_Coordinates]] + Weak Spot + Timed Review. Physics: [[Thermodynamics]] + [[Kinetic Theory]] + [[Fluids]].*

### MATH — 8:00pm to 10:10pm

**8:00–8:25 | Curvilinear: coordinate conversions**
Write all three conversion sets (polar, cylindrical, spherical) from memory. Spherical is the one people blank on — if you blank, check, rewrite three times, move on. Don't spiral. Check pass at 8:20.

*Break 8:25–8:35*

**8:35–9:00 | Curvilinear: curvature + parametric curves**
Work one curvature problem fully by hand. Then write the parametric velocity and acceleration formulas. One clean worked example for curvature is enough — don't overdo this section. Check pass at 8:55.

*Break 9:00–9:10*

**9:10–9:35 | Weak spot session**
Go back through Days 1–3 of math. Find the one specific thing that still feels unresolved — the exact step, not the whole topic. Spend this entire session only on that. Not a full re-read. Surgical. Check pass at 9:30.

*Break 9:35–9:45*

**9:45–10:10 | Timed review — one question per topic**
Six questions, one from each math topic, ~4 minutes each. No notes. Don't mark harshly — the goal is to see where your brain goes with no safety net. Check after all six. Note errors, don't fix them tonight. Check pass at 10:05.

---

### PHYSICS — 10:20pm to 11:50pm

**10:20–10:45 | Thermodynamics: first law + special processes**
Write the first law, then the four special process cases as a table — process name, what's zero, what that implies. This is a classic exam target. A self-made table beats re-reading every time. Check pass at 10:40.

*Break 10:45–10:55*

**10:55–11:20 | Thermodynamics + Kinetic Theory: heat, phase changes, gas law**
Write the Kelvin rule first — underline it. Then Q = mcΔT vs Q = Lm — know which applies when and why temperature stays flat during phase change. Then write both ideal gas law forms (moles and molecules) and the work done formulas for all three special volume cases. Check pass at 11:15.

*Break 11:20–11:30*

**11:30–11:50 | Fluids: pressure, Archimedes, Bernoulli**
For each concept write the formula, then write one real-world example next to it. The example is not decoration — it's the memory hook. Bernoulli explaining airplane lift is more memorable than the equation alone. Continuity last: narrower pipe → faster flow, same volume per second. Check pass at 11:45.

*Stop at 11:50. Sleep.*
